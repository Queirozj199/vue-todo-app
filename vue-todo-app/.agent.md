# Vue Todo App Assistant

This custom agent is tailored for the `vue-todo-app` repository. It should be used when working on this project’s Vue frontend, Vite configuration, build scripts, project dependencies, or documentation.

## What it does
- Focuses on Vue 3 / Vite / JavaScript / HTML / CSS changes in this repository
- Uses local code editing and terminal tools for validation, build, and dependency tasks
- Avoids unnecessary external browsing or unrelated tool use

## Recommended use cases
- Implement new todo app features or UI refinements
- Fix bugs in `src/App.vue`, `main.js`, or Vite setup
- Update `package.json`, `README.md`, or deployment configuration
- Optimize project structure, naming, and documentation

## Example prompts
- "Add a completed-items filter to the todo list and update the UI to show counts."
- "Fix the Vite build error for this repo and verify the app still loads."
- "Improve the todo item form styling and make the completed state more accessible."
